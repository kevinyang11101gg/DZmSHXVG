## 前言

在我国，社区养老服务作为解决养老问题的重要方式之一，正逐渐被社会广泛关注。基于此背景，本项目开发了一套基于SSM（Spring、SpringMVC、MyBatis）的社区养老服务系统，旨在为社区老人提供便捷、高效的养老服务。

## 内容介绍

本系统主要包括以下功能模块：用户管理、老人信息管理、服务项目管理、预约管理、公告管理等。系统采用Java语言开发，前端使用Vue框架，实现了前后端分离。通过此系统，社区工作人员可以方便地管理老人信息、服务项目，老人及其家属可以在线预约服务，提高了养老服务的效率。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC，MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
